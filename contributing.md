### How to Use Inpainting (Edit Specific Areas)

1. Load an image
2. Click **"Inpainting"** tab
3. Use brush tool to paint the area you want to change
4. Write what you want there: `"a red flower"`
5. Set mask blur (8-16 recommended)
6. Click **"Generate"**
7. Only masked area changes — rest stays identical

### How to Use ControlNet (Pose/Depth Control)

1. Load a reference image (pose photo, sketch, depth map)
2. Click **"ControlNet"** tab
3. Enable ControlNet
4. Choose processor:
   - **OpenPose** — for human poses
   - **Canny** — for edges
   - **Depth** — for 3D layout
5. Write your prompt matching the pose
6. Generate — output follows reference structure

### How to Install LoRA Models

1. Join community LoRA repositories
2. Download `.safetensors` LoRA file
3. Copy to `models/Lora/` folder
4. Restart Stable Diffusion AI
5. In prompt, add: `<lora:filename:0.8>`
6. Adjust weight (0.5-1.2 recommended)
7. Generate with LoRA style/character

### How to Upscale Images (High-Res Fix)

1. Generate or load an image
2. Click **"Upscale"** tab
3. Choose upscaler:
   - **4x-UltraSharp** — general use
   - **4x-AnimeSharp** — anime/cartoon
   - **ESRGAN** — realistic
4. Set scale: **2x** or **4x**
5. Enable **"Face Restoration"** if people present
6. Click **"Upscale"**
7. Wait 10-30 seconds
8. High-resolution image saved
